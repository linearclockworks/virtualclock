Install & Help

Install as an App

🍎 iPhone / iPad — Safari

  1. Open this page in Safari (not Chrome).
  2. Tap the Share button — square with arrow at the bottom.
  3. Scroll down and tap "Add to Home Screen".
  4. Tap Add.

Opens full-screen with no browser chrome, just like a native app.

🤖 Android — Chrome

  1. Open this page in Chrome.
  2. Tap the three-dot menu ⋮ top-right.
  3. Tap "Add to Home screen" or "Install app".
  4. Tap Install.

Chrome may also show an install banner at the bottom automatically.

🖥 Desktop — Chrome or Edge

  1. Open this page in Chrome or Edge.
  2. Click the install icon ⊕ in the address bar.
  3. Click Install.

The clock opens in its own window and appears in your dock or taskbar.

Controls

  1. Demo — animates the pointer across the full day.
  2. Speed — demo animation speed; drag to zero to pause.
  3. 🌙 — night-mode dim level.
  4. 🕒 / 🔢 — toggle analog / digital reference clocks.
  5. Details — opens the product page for this clock.
  6. ☕ — support Linear Clockworks on Ko-fi.
